# AgentMail: Email for AI Agents

## What is AgentMail?

[AgentMail](https://www.agentmail.to) is an API-first email platform designed specifically for building AI agents that can communicate over email. It gives agents their own email inboxes, like Gmail has for humans.

## Why We Use It

Instead of fighting with browser automation and DOM issues (like Gmail's Shadow DOM), AgentMail gives us a clean API to send/receive emails.

### Problems Solved
- ❌ Gmail browser automation fails with Shadow DOM
- ❌ Button clicks don't work reliably  
- ❌ Email sending requires complex workarounds

- ✅ AgentMail: Simple API call to send emails
- ✅ Reliable, programmatic email access
- ✅ No browser needed
